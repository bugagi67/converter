# Универсальный конвертер прайсов

Профессиональный инструмент для конвертации любых прайс-листов в единый JSON/CSV/XLSX формат.

Поддерживает:
- CSV
- XLSX / XLS
- XML
- YML / YAML (Яндекс.Маркет, 1С и др.)

Автоматически распознаёт поля: артикул, цена, остаток, бренд, изображения и т.д.  
Работает с файлами до 100 000+ товаров без зависаний благодаря прогресс-бару и chunk-обработки.

---

### Особенности

- Загрузка по прямой ссылке (через CORS-прокси)
- Drag & Drop + выбор файла
- Умный маппинг полей (расширяемый)
- Прогресс-бар с процентами и анимацией
- Скачивание результата в 3 форматах: JSON, CSV, XLSX
- Предпросмотр результата
- Поддержка `param_Поле` из YML
- Показ неизвестных полей (для доработки маппинга)

### Установка и запуск (Vite + React)

```bash
# Клонируй репозиторий
git clone https://github.com/yourname/price-converter.git
cd price-converter

# Установи зависимости
npm install
# или
yarn
# или
pnpm install

# Запусти в режиме разработки
npm run dev
```

### Используемые технологии

- React 18
- Vite (быстрый старт и HMR)
- SheetJS (xlsx) — парсинг Excel и CSV
- Нативный DOMParser — для XML/YML
- Zero dependencies для UI (только inline-стили)
